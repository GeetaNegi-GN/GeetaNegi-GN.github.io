# Privacy Policy for NotiCatch

Last updated: June 2, 2026

This Privacy Policy explains how NotiCatch, published by Geeta Negi, handles information when you use the mobile application.

NotiCatch is designed to help users organize notifications, manage notification history, filter alerts, create reports, and improve notification control.

By using NotiCatch, you agree to this Privacy Policy.

## 1. Information We Access

NotiCatch may access the following information depending on the permissions you enable:

### Notification Information

When Notification Access permission is enabled, NotiCatch may read notification data from apps selected by you, including:

* App name
* Notification title
* Notification message/content
* Notification time
* Notification category information

This access is required for core features such as:

* Notification history
* Priority organization
* Notification categories
* Smart Quiet Mode
* Saving important notifications
* Exporting notification reports

### Device Information

Basic technical information may be used for normal app functionality, such as:

* Device type
* Android version
* App version

## 2. How Information Is Used

Information is used only to provide NotiCatch features, including:

* Displaying notification history
* Organizing notifications into categories
* Highlighting important notifications
* Filtering low-priority alerts
* Providing custom notification alerts
* Creating exported reports
* Backup and restore functionality

## 3. Notification Access Permission

NotiCatch requires Android Notification Access permission to read and organize notifications.

You control this permission and can enable or disable it anytime from Android system settings.

NotiCatch only processes notifications from apps selected by you inside the application.

## 4. Data Storage

Notification information is stored locally on your device.

NotiCatch does not sell your notification data.

Users can delete stored notification history anytime using app controls.

## 5. Backup, Import, and Export

NotiCatch allows users to:

* Export notification reports
* Create backup files
* Import backups on another device

Backup files are created only when requested by the user.

Users are responsible for managing and sharing exported files.

## 6. Data Sharing

NotiCatch does not sell personal information.

Your notification content is not shared with third parties for advertising purposes.

Information may only be processed when necessary to:

* Provide app functionality
* Comply with legal requirements
* Maintain app security

## 7. Third-Party Services

NotiCatch may use trusted third-party services provided by platforms such as:

* Google Play services
* Android system services

These services may process limited technical information according to their own privacy policies.

## 8. Data Security

Reasonable measures are used to protect stored information.

However, no electronic storage method can be guaranteed to be completely secure.

## 9. Children's Privacy

NotiCatch is not designed to knowingly collect personal information from children.

If you believe a child has provided information improperly, please contact us so it can be reviewed.

## 10. Your Control

You can:

* Disable Notification Access permission anytime
* Choose which apps NotiCatch monitors
* Delete stored notifications
* Remove exported files
* Uninstall the app to remove locally stored data

## 11. Changes to This Policy

This Privacy Policy may be updated when app features or requirements change.

The latest version will always be available through the official privacy policy link.
